How AI Was Used

AI tools were used to generate the HTML/CSS/JS structure, the probability math functions (PDF evaluation, numerical integration, parameter estimation routines), the interactive canvas drawing logic, and the histogram overlay code for C9.

What Was Verified

All probability results were checked against known analytic values: Uniform P = (x₂−x₁)/(b−a); Normal 68–95–99.7 rule (P(μ±σ) ≈ 0.6827); Triangular CDF closed-form formula. PDFs were verified to integrate numerically to 1 ± 0.001.

Testing

Each distribution was tested with edge cases: bounds equal to parameters, very small σ for the normal, single-piece piecewise. The C9 tool was tested with three datasets: one uniform-like (evenly spread over [0,10]), one triangular-like (peak near 5), and one normal-like (μ≈5, σ≈1.5).
